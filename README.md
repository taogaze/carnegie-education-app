# 卡内基教育小程序

一个基于Vue 3的教育类小程序，专注于沟通技巧和领导力培训。

## 项目特色

- 🎨 现代化UI设计，移动端优先
- 📱 响应式布局，适配各种屏幕尺寸
- ⚡ 基于Vue 3 + Vite，开发体验优秀
- 🎯 专注教育内容，用户体验友好
- 🌈 精美的渐变色彩和动画效果

## 项目结构

```
卡内基训练/
├── src/
│   ├── components/          # Vue组件
│   │   ├── Banner.vue      # 顶部横幅组件
│   │   ├── CourseList.vue  # 课程列表组件
│   │   └── TabBar.vue      # 底部标签栏组件
│   ├── styles/             # 样式文件
│   │   └── main.css        # 主样式文件
│   ├── App.vue             # 根组件
│   └── main.js             # 应用入口
├── index.html              # HTML模板
├── package.json            # 项目配置
├── vite.config.js          # Vite配置
└── README.md               # 项目说明
```

## 功能模块

### 1. Banner组件
- 渐变背景设计
- 动态旋转动画效果
- SVG图标集成
- 响应式文字布局

### 2. 课程列表组件
- 课程卡片展示
- 价格信息显示
- 课程难度标识
- 悬停交互效果

### 3. 底部标签栏组件
- 四个主要导航标签
- SVG图标支持
- 活跃状态切换
- 点击动画效果

## 技术栈

- **前端框架**: Vue 3
- **构建工具**: Vite
- **样式方案**: CSS3 + Scoped CSS
- **图标方案**: SVG
- **开发语言**: JavaScript

## 快速开始

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

### 预览生产版本
```bash
npm run preview
```

## 设计理念

### 色彩方案
- 主色调：渐变紫蓝色 (#667eea → #764ba2)
- 辅助色：各种鲜艳的课程标识色
- 背景色：浅灰色 (#f5f5f5)

### 交互设计
- 卡片悬停效果
- 按钮点击反馈
- 平滑过渡动画
- 触摸友好的交互区域

### 响应式设计
- 移动端优先设计
- 适配320px-768px屏幕
- 弹性布局和网格系统
- 触摸设备优化
